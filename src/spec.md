# Haya Attar and Perfume Showcase Website

## Overview
A luxury perfume showcase website for the Haya Attar and Perfume brand, featuring an elegant design that emphasizes fragrance aesthetics and brand sophistication.

## Core Features

### Homepage
- Brand introduction section with elegant hero area
- Brief description of Haya Attar and Perfume brand
- Navigation to other sections of the website
- Luxury aesthetic with modern, clean design

### Perfume Gallery
- Display collection of perfumes with visual cards
- Each perfume entry includes:
  - Product image
  - Perfume name
  - Detailed scent description
  - Key fragrance notes
- Grid or card-based layout for easy browsing
- Responsive design for all devices

### Contact Section
- Company contact information display
- Social media links and icons
- Professional contact details presentation
- Clean, accessible layout

## Backend Requirements
- Store perfume data including names, descriptions, and image references
- Provide API endpoints to retrieve perfume collection data
- Store contact information and social media links

## Design Requirements
- Modern, luxury aesthetic
- Emphasis on visual elegance and sophistication
- Color scheme appropriate for fragrance/perfume industry
- Clean typography and spacious layouts
- Mobile-responsive design
- Professional imagery integration

## Content Language
- All content in English
